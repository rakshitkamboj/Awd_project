JEE Main Rank Predictor

JEE Main Rank Predictor 2021 is a tool to predict JEE main rank in 2021. The rank is predicted based on marks vs percentile vs rank information collected from 2017 to 2020. For details on how these data points are used to predict JEE Main Rank do visit college predictor dot [AI].We also have a set after the JEE exam tools. You might be interested to take a look at JEE Main College Predictor, JEE Advance Rank Predictor and JEE advanced College Predictor. The College predictors give a list of rank wise college lists for different categories, rounds, and years. Do take a look and don’t forget to share it with friends.


JEE Main Rank Predictor: Features

  1. The Rank predictor will predict the expected rank for JEE main 2020 for expected marks in the mock test or final JEE main exam.
  2. Build on a solid mathematical foundation.
  3. Predict JEE Main ranks for all categories (Open, BC, SC, ST, and yes even EWS).
  4. The highest marks in JEE Main 2017 were 350 and for JEE main 2018 it was for 360 marks. Don’t forget to normalize them to 300 marks if you want to verify old scores.
  5. A range of Ranks is provided for available adjacent marks and the final rank is predicted based on linear interpolation.
  6. The approximate number of candidates for given marks are calculated based on 2018, 2018, 2019 data.
